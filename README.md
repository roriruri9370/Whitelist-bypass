### Ссылка для приложения:
`https://raw.githubusercontent.com/DarkFirexs/-VPN/refs/heads/main/Whitelist%20%7C%20VPN`

---

### Что скачивать:
* **v2rayTun** — для обычного использования.
* **v2rayNG** — если нужно просто потестить сервера.

### Как настроить:

**В v2rayTun:**
1. Скопируй ссылку выше.
2. Нажми на **"+"** и выбери **"Импорт из буфера обмена"** (или "Ручной ввод" и вставь ссылку в поле URL).
3. Пользоваться.

**В v2rayNG:**
1. Скопируй ссылку.
2. Нажми на три точки в углу справа, **"Добавить профиль"** ➔ **"Импорт из буфера обмена"**.
